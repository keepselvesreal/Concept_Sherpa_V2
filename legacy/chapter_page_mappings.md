## 검증 방법

각 챕터의 시작을 다음과 같이 확인했습니다:

1. **패턴 매칭**: 챕터 제목과 부제목 패턴 검색
2. **내용 확인**: "This chapter covers" 등 챕터 시작 지표 확인
3. **텍스트 길이**: 충분한 양의 실제 내용 존재 확인
4. **목차 제외**: 목차 페이지가 아닌 실제 내용 페이지인지 확인
